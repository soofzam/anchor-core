Contributing to Anchor Core

Anchor Core is intentionally open to critique, replication, and adversarial review.

This project does not assume correctness. It assumes iteration.

Ways to Contribute

Replicate the benchmark with additional models

Propose alternative scoring methods

Add new stress prompts

Challenge failure classifications

Document counterexamples

Suggest formalization improvements

Methodological Transparency

When submitting changes or critiques, please include:

Exact prompt wording

Model version (if known)

Full model output

Scoring rationale

Any external verification steps taken

Tone Expectations

Anchor is a standards discussion.

Disagreement is expected. Personal attacks are not.

Strong critiques improve the standard.
